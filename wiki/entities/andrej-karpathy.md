# Andrej Karpathy

## Overview

Co-founder of OpenAI, former Tesla AI director, and widely recognized as one of the clearest explainers of machine learning concepts. Creator of the LLM Wiki pattern for personal knowledge management.

## Key Information

- **Roles**: 
  - Co-founder of OpenAI
  - Former Director of AI at Tesla
  - AI educator and thought leader
- **Notable contributions**:
  - LLM Wiki pattern/methodology (announced via tweet April 2, 2026)
  - GitHub gist "llm-wiki.md" — "idea file" describing the pattern
  - Clear communication of complex ML concepts
  - Advocacy for structured, LLM-readable knowledge systems
- **Philosophy**: 
  - Organize information for AI consumption rather than just human navigation
  - Promote practical, simple patterns over complex infrastructure
  - Software compilation analogy: "treat knowledge the same way as code — compile once, run many times"
- **Communication style**: Shares concepts as "idea files" meant to be copy-pasted into LLM agents for instantiation, not as rigid implementations
